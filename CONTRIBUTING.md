# Contributing

Pull requests for bug fixes are always welcome!

Before submitting new features or changes to current functionality, it is recommended to first
[open an issue](https://github.com/open-telemetry/opentelemetry-ecosystem-explorer/issues/new)
and discuss your ideas or propose the changes you wish to make.

## Style Guide

Run the following checks to ensure your code meets the project's standards:

```bash
# Linter check
npm run lint:md

# to fix
npm run lint:md:fix
```
